Seviye 1: İlk C Programımız

C dili, 1972 yılında Dennis Ritchie tarafından geliştirilmiş güçlü bir programlama dilidir. Her C programı main() fonksiyonu ile başlar.

// İlk C programımız #include <stdio.h> int main() { printf("Merhaba Dünya!"); return 0; }

İlk hazinenizi bulmak için sayfanın CSS kodlarını incelemelisiniz.

Tarayıcınızda F12 tuşuna basarak başlayın ve CSS kısmına bakın.

İpucu: C programlarının kalbi olan fonksiyon ismini arıyoruz!

CSS kodlarında yorum satırları arasında gizlenmiş bir şifre var. C dilinde her programın başladığı fonksiyonun adıyla ilgili.
Seviye 2: Değişkenler ve Veri Tipleri

Tebrikler! C dilinde değişkenler veri saklamak için kullanılır. Her değişkenin bir veri tipi vardır.

int sayi = 42; // Tam sayı float ondalik = 3.14; // Ondalıklı sayı char harf = 'A'; // Karakter

Bu sayfada bir yerde gizli bir metin var. Onu bulmak için fare imlecini sayfada gezdirmelisin.

İpucu: Bazen şeffaf metinler fare ile üzerlerine gelindiğinde görünür olur!

C dilinde temel veri tipleri şunlardır: Gizli mesajı buldun! İkinci hazinenin şifresi: "data-types" int, float, double, char. Bu tipler farklı boyutlarda bellek kullanır.

Sayfadaki bazı metinler şeffaf olabilir, üzerlerine mouse ile gelince görünür olurlar.
Seviye 3: Döngüler (Loops)

Harika ilerliyorsun! C dilinde döngüler, aynı kodun tekrar tekrar çalıştırılması için kullanılır.

// For döngüsü örneği for(int i = 0; i < 10; i++) { printf("%d ", i); }

Üçüncü hazine çok ama çok küçük bir metin olarak sayfanın bir yerinde saklanmış.

İpucu: Bazen küçük şeylerin üzerine geldiğinde büyürler!

C dilinde üç tip döngü vardır:

  • for: Belirli sayıda tekrar için
  • while: Koşul doğru olduğu sürece
  • do-while: En az bir kez çalışır
Üçüncü hazinenin şifresi: "for-while"
Sayfada fare imlecini her yere götür ve değişiklik olan yerlere dikkat et. Çok küçük görünmez şeyler büyüyebilir!
Seviye 4: Koşullu İfadeler

Çok iyisin! C dilinde koşullu ifadeler, programın farklı yollar izlemesini sağlar.

if(sayi > 0) { printf("Pozitif sayı"); } else if(sayi < 0) { printf("Negatif sayı"); } else { printf("Sıfır"); }

Dördüncü hazine JavaScript konsol mesajları arasında gizli!

İpucu: F12 tuşuna bas ve "Console" sekmesine tıkla, sonra aşağıdaki butona tıkla.

Console sekmesinde görünen mesajları oku, dördüncü hazine orada yazıyor!
Seviye 5: Fonksiyonlar

Mükemmel! Son seviyeye ulaştın. C dilinde fonksiyonlar, kodun organize edilmesi ve tekrar kullanılması için önemlidir.

int topla(int a, int b) { return a + b; } int main() { int sonuc = topla(5, 3); printf("%d", sonuc); return 0; }

Son hazine bir JavaScript fonksiyonunun içinde gizli. Konsolu açmalısın!

İpucu: Aşağıdaki butona tıkla ve konsol mesajlarını kontrol et.

Console sekmesinde görünen son mesajı oku, beşinci hazine orada yazıyor!
